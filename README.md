# UV IoT Device

Project to make a battery powered IoT device for the measurement of UV radiation, temperature and visible light levels.

### Python Requirements

- Python>=3.7
- RPi.GPIO>=0.7
- gpiozero>=1.5
- numpy>= 1.16

### Hardware Requirements

- Raspberry Pi Zero W
- Thermistor
- PCF8591 (or another ADC)
- 5000 or 10000 mAh powerbank
